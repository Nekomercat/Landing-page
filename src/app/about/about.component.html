La Estación Meteorológica consta de una app de python que muestra gráficas de la temperatura, la humedad del suelo y la cantidad de luz en el ambiente. Para esto, se utilizan sensores que envían señales analógicas a un microcontrolador ESP32. Utilizando el IDE de Arduino, el microcontrolador convierte las tres señales a digitales y envía los datos por internet utilizando una API desarrollada en GraphQL. El front-end de la página está desarrollado con Angular y adopta las funcionalidades de iniciar sesión o crear una nueva cuenta con el fin de obtener un token para poder ingresar nuevos datos a la base.
